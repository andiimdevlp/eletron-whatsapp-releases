# Mass Sender Releases

Este repositório **existe exclusivamente** para hospedar os arquivos de atualização automática da aplicação **Mass Sender** (WhatsApp Mass Sender).

### O que você encontra aqui
- Arquivos de instalação (.exe para Windows)
- Arquivos de metadados para auto-update (`latest.yml`, `.blockmap`, etc.)
- Releases geradas automaticamente pelo processo de build

### Importante
- **Este repositório não contém código-fonte**  
- O código fonte completo e privado está em:  
  https://github.com/andiimdevlp/eletron-whatsapp

- As atualizações são distribuídas automaticamente pela aplicação para usuários autorizados.  
- Não é necessário clonar ou baixar nada manualmente daqui — a aplicação cuida disso.

Qualquer dúvida sobre a aplicação, entre em contato com o desenvolvedor.
